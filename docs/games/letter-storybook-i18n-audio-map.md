## Canonical key naming (important for DUB-657)

To align with existing cross-game letter packs, the storybook uses the existing canonical letter key set:

`alef, bet, gimel, dalet, he, vav, zayin, het, tet, yod, kaf, lamed, mem, nun, samekh, ayin, pe, tsadi, qof, resh, shin, tav`

Note: use `qof` (not `kuf`) for consistency with `common.letters.*` families.

## Association-word alignment (DUB-654 consistency)

Storybook letter associations are aligned to the canonical DUB-654 lexicon and current `common.letters.anchorWords.*` usage to avoid cross-game vocabulary drift.
